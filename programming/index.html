Software packages

The software packages I have written or are currently working on are listed below:

Q-Tensor-3d

I wrote this software package to solve the Landau-de Gennes free energy problem. The software is written in Python, and uses the package Firedrake, an open source FEM package likewise written in Python.

GitHub: https://github.com/andrewlhicks/q-tensor-3d

SymPyPlus

This package extends SymPy to be able to do calculus of variations. I use this in conjunction with Q-Tensor-3d.

GitHub: https://github.com/andrewlhicks/sympyplus
